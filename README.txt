شرح سريع لتشغيل التطبيق PWA على iPad / جهاز آخر:

1) انسخ مجلد المشروع إلى استضافة تدعم HTTPS (مثلاً GitHub Pages, Netlify, Vercel).
2) افتح الرابط في متصفح Safari على iPad.
3) لإضافة التطبيق إلى الشاشة الرئيسية: زر المشاركة -> Add to Home Screen.
4) الدخول إلى لوحة الإدارة: اضغط "دخول" ثم أدخل PIN الافتراضي 1234.

ملاحظات:
- التطبيق يخزن البيانات محليًا (localStorage). إذا أردت مشاركة المواعيد بين أجهزة أو إرسال تذكيرات SMS/Push، سأجهز لك backend (Node/SQLite أو Firebase).
- أضف ملفات الأيقونات icon-192.png و icon-512.png إلى المجلد قبل النشر.
